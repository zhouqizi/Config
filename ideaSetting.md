**代码补全设置**

如下图，进入设置菜单Settings | Editor | General | Code Completion：

①代码补全时，匹配候选代码忽略大小写，去掉勾选Match case。

②代码补全时，勾选Show the documentation popun自动弹出方法用法文档(且等待时间缩短至100毫秒)

③代码补全，方法参数信息提示，三个选择都勾上(且等待时间缩短至100毫秒)，这样可类似于Eclipse提示方法的参数。

设置后需要点“Apply”按钮生效。
